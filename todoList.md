## Checklist: Trabalho 2 (React)

### Requisitos Gerais e Tecnologias Base

[ ] Definir o tema livre da aplicação web.

[ ] Implementar a aplicação client-side/front-end com React.

[ ] Manter os requisitos do Trabalho 1: Git, Github, HTML 5, CSS, Bootstrap e JavaScript.

#### Requisitos Específicos de React

[ ] Componentes: Criar pelo menos 3 componentes (além do App.js).

[ ] JSX: Executar códigos JavaScript dentro do JSX.

[ ] Interpolação: Interpolar pelo menos 3 tipos de variáveis (comuns, arrays, objetos).

[ ] Eventos: Utilizar pelo menos 3 tipos de eventos (ex: mouse, teclado, formulário, arrasto, transferência, mídia entre outros).

[ ] Imagens: Importar e exibir uma imagem.

[ ] useState: Utilizar useState para pelo menos 3 tipos de variáveis (comuns, arrays, objetos).

[ ] Renderização Condicional: Implementar renderização condicional usando useState.

[ ] Props: Utilizar pelo menos 3 tipos de props (variáveis comuns, variáveis de estado, função).

[ ] Reuso de Componente: Reutilizar um componente através de loop.

[ ] Formulário: Criar um formulário com pelo menos 5 tipos de input (ex: text, password, email, checkbox, select, textarea, number, radio, datalist, file, entre outros).

#### CSS (Estilização Dinâmica)

[ ] Classe Dinâmica: Utilizar pelo menos 1 classe CSS dinâmica.

[ ] CSS Modules: Utilizar pelo menos 1 CSS Module.

#### Requisições HTTP

[ ] API: Criar (simular) ou utilizar uma API pronta.

[ ] Fetch API: Utilizar a Fetch API.

[ ] useEffect: Utilizar o hook useEffect.

[ ] Custom Hook: Criar um custom hook para realizar requisições (GET/POST).

Entrega e Apresentação

[ ] Documentação: Documentar o código seguindo boas práticas.

[ ] Entrega: Enviar o link do repositório público no GitHub para o professor (via Teams).
